# Manual Testing Project – Demo Web Shop

## Project Overview

This project demonstrates manual testing activities performed on the Demo Web Shop website.

The objective is to practice Software Testing Life Cycle (STLC), Test Case Design, Bug Reporting, and Test Summary Reporting.

---

## Modules Tested

* Register
* Login
* Search
* Shopping Cart
* Checkout

---

## Testing Deliverables

### Test Plan

* Define testing scope and strategy.
* Identify test objectives and schedule.

### Test Cases

* Total Test Cases: 90
* Positive Testing
* Negative Testing
* Boundary Testing
* Validation Testing

### Bug Reports

* Total Bugs: 10
* Severity Classification
* Priority Classification

### Test Summary Report

* Execution Result
* Defect Summary
* Test Conclusion
